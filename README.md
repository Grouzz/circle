# circle of life

## Requirements
- Python
- Dependencies:
  ```bash
  pip install pygame
  ```

## Assets (optional)
place images next to `main.py` :
- `grass.png`
- `prey.png`
- `prey_active.png`
- `predator.png`

If an image is missing, the program uses colored tiles instead.

## Run
```bash
python main.py
```

## Controls
- `R` : toggle rain (faster grass growth)
- `SPACE` : toggle drought (stops grass growth)
- `ESC` : quit
